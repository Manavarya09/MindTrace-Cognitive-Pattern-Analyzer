# MindTrace - Cognitive Pattern Analyzer

## Overview
MindTrace is an advanced cognitive pattern analysis system that analyzes personal text data to discover patterns in thinking style, emotional trends, productivity cycles, and behavioral insights.

## Key Features
- Sentiment Analysis (VADER + TextBlob)
- Emotion Detection (6 emotions)
- Productivity Pattern Detection
- Behavioral Trend Analysis
- Semantic Clustering
- Interactive Visualizations
- Automated Insights
- Cognitive Reports

## Quick Start
```python
# Run analysis
dashboard.run_analysis(n_samples=1000)

# View results
dashboard.show_visualizations()
dashboard.generate_report()
```

## Requirements
- Python 3.8+
- pandas, numpy
- nltk, spacy, textblob
- scikit-learn
- plotly, wordcloud
- jupyter
